Command to download openApiGenerate jar file:
wget https://repo1.maven.org/maven2/org/openapitools/openapi-generator-cli/7.17.0/openapi-generator-cli-7.17.0.jar -O openapi-generator-cli.jar
OpenApiGenerator in C#/.NET require Java to be installed on your machine.

Linki do dokumentacji OpenAPI Generator:
https://github.com/OpenAPITools/openapi-generator/tree/master
https://openapi-generator.tech/docs/globals

Command to generate api client:
java -jar openapi-generator-cli.jar generate -i ./ShopWeb.Infrastructure/ApiClient/shopApi.json -g csharp -o ./ShopWeb.Infrastructure/ApiClient/OpenApiGenerate/Infrastructure/ShopApiClient -c ./ShopWeb.Infrastructure/ApiClient/OpenApiGenerate/openApiConfig.json

Command to generate openApiGenerate client when all configuration are inside openApiConfig.json:
java -jar openapi-generator-cli.jar generate -c ./ShopWeb.Infrastructure/ApiClient/openApiConfig.json

!!!If you have installed Java and downloaded openapi-generator-cli.jar file you can run only this script to generate client in PackageManagerConsole in Infrastructure project!!!
This script generate api client, copy models, api classes and all infrastructure project to appropriate folders with changing the namespaces. 
It also delete unnecessary files generated by OpenAPI Generator.
